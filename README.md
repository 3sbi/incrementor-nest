## Description

framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# watch mode
$ npm run watch

# production mode
$ npm run prod
```

or using docker

```bash
$ docker compose up
```
